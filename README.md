# Badminton Cost Split

Simple browser app to split badminton session costs.

## Features

- Inputs:
  - Number of players (2–40, default 6)
  - Shuttlecocks used (0–40, default 3)
  - Price per shuttlecock (default 90)
  - Court price (default 440)
- Calculates:
  - Total cost
  - Cost per player
- Saves last entered values to localStorage
- Includes favicon + PWA-style icons for mobile home screen

## Run locally

Just open `index.html` in your browser.

## Deploy

Can be hosted as static files (GitHub Pages recommended).
